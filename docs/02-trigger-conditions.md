# 02 · 觸發條件

## 入口觸發規則

| 條件 | 說明 |
|------|------|
| 入口位置 | 【交誼廳】→【獲取寶石】頁籤 |
| 觸發方式 | 玩家點擊進入 |
| 前置條件 | 玩家已進入交誼廳 |

## 遊玩條件

| 條件項目 | 說明 |
|----------|------|
| 金幣餘額 | 玩家金幣餘額須 ≥ 當前押注額，否則無法開始遊玩 |
| 每日可遊玩次數 | 剩餘次數 > 0 才可遊玩 |
| 非轉動中 | 轉盤未在轉動狀態才可再次啟動 |

## VIP 每日可遊玩次數（依國家與 VIP 等級）

每日可旋轉次數依玩家 VIP 等級而定。VIP 1（銅）和 VIP 2 不可遊玩，VIP 3 以上開放。

| VIP 等級 | 名稱 | 每日可遊玩次數 | 狀態 |
|----------|------|---------------|------|
| 1 | 銅 | 0（不能玩） | ✅ 已確認 |
| 2 | — | 0（不能玩） | ✅ 已確認 |
| 3 | — | 20 | ✅ 已確認（與影片觀察值吻合） |
| 4 | — | 50 | ✅ 已確認 |
| 5 | — | 100 | ✅ 已確認 |
| 6 | 鑽石 | 150 | ✅ 已確認 |

> **備註**：影片中觀察到每日上限為 20 次（顯示為「今日剩餘旋轉 20/20 次」），對應 VIP 3。規則頁面會顯示各 VIP 的可旋轉次數。國家維度的限制如有差異，由後臺 `daily_spin_limit` 參數另行配置。

## 排除與限制條件

| 條件 | 說明 |
|------|------|
| 次數歸零 | 當日可遊玩次數歸零後，進入冷卻倒數，期間無法遊玩 |
| 冷卻期間 | 【開始】按鈕文案變更並壓暗不可點；+/- 按鈕壓暗不可點 |
| 金幣不足 | 金幣餘額不足當前押注額時，無法開始遊玩；彈出 Toast 提示「金幣不足」（⚠️ 建議值，提示樣式待 UI 確認） |

## 可調參數

| 參數名稱 | 說明 | 狀態 |
|----------|------|------|
| `daily_spin_limit` | 每日可遊玩次數（依 VIP 等級）：VIP1=0, VIP2=0, VIP3=20, VIP4=50, VIP5=100, VIP6=150 | ✅ 已確認 |
| `bet_options` | 可用押注額選項列表 | ✅ 已確認：50,000,000 / 100,000,000 / 500,000,000 |
| `cooldown_duration` | 次數歸零後的冷卻倒數時間 | ⚠️ 建議值：等待至伺服器午夜 00:00 重置（依按鈕 RESET 文案推論） |
| `auto_spin_options` | 自動旋轉手數選項 | ✅ 已確認：10 / 50 / 100 / ∞ |
| `long_press_threshold` | 長按觸發自動旋轉的時間閾值 | ✅ 已確認（約 2 秒） |
