# 02 · 觸發條件

## 入口觸發規則

| 條件 | 說明 |
|------|------|
| 入口位置 | 【交誼廳】→【獲取寶石】頁籤 |
| 觸發方式 | 玩家點擊進入 |
| 前置條件 | 玩家已進入交誼廳 |

## 遊玩條件

| 條件項目 | 說明 |
|----------|------|
| 金幣餘額 | 玩家金幣餘額須 ≥ 當前押注額，否則無法開始遊玩 |
| 每日可遊玩次數 | 剩餘次數 > 0 才可遊玩 |
| 非轉動中 | 轉盤未在轉動狀態才可再次啟動 |

## VIP 每日可遊玩次數（依國家與 VIP 等級）

每日可旋轉次數依玩家 VIP 等級和所在國家而定。

| VIP 等級 | 每日可遊玩次數 | 狀態 |
|----------|---------------|------|
| （影片中觀察值） | 20 次 | ✅ 已確認（影片實測，該測試帳號為此值） |
| 完整 VIP × 國家矩陣 | **TBD** | 🔴 待確認（由數值企劃設定） |

> **備註**：影片中觀察到每日上限為 20 次（顯示為「今日剩餘旋轉 20/20 次」）。完整的次數規則為分國家與 VIP 等級的矩陣，詳細數值由數值企劃提供。規則頁面會顯示各 VIP 的可旋轉次數。

## 排除與限制條件

| 條件 | 說明 |
|------|------|
| 次數歸零 | 當日可遊玩次數歸零後，進入冷卻倒數，期間無法遊玩 |
| 冷卻期間 | 【開始】按鈕文案變更並壓暗不可點；+/- 按鈕壓暗不可點 |
| 金幣不足 | 金幣餘額不足當前押注額時，無法開始遊玩（具體提示行為 **TBD**） |
