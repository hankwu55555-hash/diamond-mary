# 04 · 介面顯示

## 主畫面 UI 元素清單

| 編號 | UI 元素 | 說明 | 字數/位元顯示上限 |
|------|---------|------|-------------------|
| 1.0 | 資產欄位 | 顯示玩家的金幣與寶石，左側金幣／右側寶石，常駐於畫面上方 | 金幣：15 位元（超過 12 位元縮小顯示）；寶石：15 位元（超過 12 位元縮小顯示） |
| 2.0 | 小瑪莉輪盤 | 主遊戲轉盤，玩家使用金幣押注換取寶石 | 各格子數字依押注額連動 |
| 3.0 | 押注區 | 顯示當前押注額，含 +/- 切換按鈕 | 金幣圖案 + 9 位元 |
| 4.0 | 【開始遊玩/領取】按鈕 | 短按開始遊玩；長按約 2 秒觸發自動旋轉；結果確定後變為【領取】 | — |
| 5.0 | 規則 | 顯示玩家當前 VIP 的各等級可遊玩次數 | — |
| 6.0 | 【關閉】按鈕 | 按下後關閉介面 | — |

## 轉盤版面規格

| 位置 | 視覺規格 |
|------|----------|
| 上方 | 猴爺圖 + 寶石數量框 |
| 下方 | 聚寶盆圖 + 寶石數量框 |
| 四角 | 紫底格子 |
| 其他格子 | 紅底或青底交替 |

> 切換押注額時，轉盤上各格子的數字同步變化，並根據位數縮放字體大小。

## 按鈕狀態對照

### 【開始遊玩/領取】按鈕

| 狀態 | 按鈕文案 | 按鈕行為 | 視覺表現 |
|------|----------|----------|----------|
| 閒置可遊玩 | START（開始） | 短按 → 開始遊玩；長按約 2 秒 → 跳出自動旋轉面板 | 正常可點擊狀態 |
| 轉盤轉動中 | — | 不可點擊 | 壓暗 |
| 結果確定 | COLLECT（領取） | 點擊 → 寶石飛入資產欄位 | 可點擊狀態 |
| 自動旋轉中 | 依選擇變化（顯示剩餘手數） | 點擊 → 停止自動旋轉 | 顯示 STOP（停止） |
| 次數歸零 | 冷卻文案 + 倒數 | 不可點擊 | 壓暗 + 顯示冷卻倒數 |

### +/- 押注切換按鈕

| 狀態 | 行為 |
|------|------|
| 正常 | 按 + 切到下一個押注額，按 - 切到前一個押注額 |
| 到達最大額按 + | 循環跳至最小押注額 |
| 到達最小額按 - | 循環跳至最大押注額 |
| 轉盤轉動中 | 不可點擊 |
| 次數歸零 | 壓暗不可點 |

## 自動旋轉選擇面板

| 項目 | 規格 |
|------|------|
| 觸發方式 | 長按【開始遊玩】約 2 秒 |
| 選項數量 | 最多 4 個 |
| 選項內容 | 包含固定手數選項與【無限】選項（具體選項值 **TBD**） |
| 面板行為 | 不可滑動；點擊任一選項或外部位置皆會關閉面板 |
| 選擇後 | 按鈕文案根據選擇變化；非【無限】選項每手遞減顯示 |
