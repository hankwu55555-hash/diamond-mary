# 01 · 基礎規格

## 系統概述

「寶石小瑪莉」是一款嵌入於交誼廳的小瑪莉輪盤遊戲。玩家使用金幣押注，透過橢圓形轉盤的燈號循環跑動獲取寶石獎勵。遊戲沿用傳統小瑪莉的轉盤版面設計，上方為猴爺圖搭配寶石數量框、下方為聚寶盆圖搭配寶石數量框。格子分為紫底六角形（大格，高獎勵）與綠底/紅底圓形（一般格）。每日可遊玩次數依玩家的 VIP 等級與所在國家而定。
