# 01 · 基礎規格

## 系統概述

「寶石小瑪莉」是一款嵌入於交誼廳的小瑪莉輪盤遊戲。玩家使用金幣押注，透過橢圓形轉盤的燈號循環跑動獲取寶石獎勵。遊戲沿用傳統小瑪莉的轉盤版面設計，上方為猴爺圖搭配寶石數量框、下方為聚寶盆圖搭配寶石數量框。格子分為紫底六角形（大格，高獎勵）與綠底/紅底圓形（一般格）。每日可遊玩次數依玩家的 VIP 等級與所在國家而定。

## 核心規則表

| 規則項目 | 說明 |
|----------|------|
| 入口位置 | 玩家從【交誼廳】的【獲取寶石】頁籤點入 |
| 押注機制 | 使用 +/- 按鈕循環切換押注額（50M → 100M → 500M → 50M…） |
| 轉盤連動 | 切換押注額時，轉盤上 22 格的寶石數字等比同步變化，並依位數縮放字體 |
| 遊玩流程 | 點擊【開始】→ 金幣立即扣除 → 燈號循環跑動 → 停止後選中格子高亮 → 點擊【領取】獲得寶石 |
| 自動旋轉 | 長按【開始】約 2 秒，跳出選擇面板（10 / 50 / 100 / ∞），選擇後自動遊玩 |
| 自動旋轉領取 | 自動旋轉期間不需按【領取】，寶石自動飛入並顯示「+X,000」浮動提示 |
| 非無限自動旋轉 | 選擇非【∞】選項時，按鈕上顯示剩餘手數並每手遞減 |
| 次數歸零 | 可遊玩次數歸零時，【開始】按鈕壓暗不可點，顯示重置倒數；+/- 按鈕同時壓暗 |
| 金幣不足 | 金幣餘額 < 當前押注額時，點擊【開始】彈出提示（詳見 [04-介面顯示](04-ui-display.md)） |
| 遊玩中操作限制 | 燈號跑動期間，轉盤區域、押注區、【開始】按鈕不可點擊 |
| 隨時關閉 | 遊玩過程中可點擊【關閉】直接關閉交誼廳畫面 |

## 獎勵結構

| 獎勵類型 | 說明 | 格數 | 備註 |
|----------|------|------|------|
| 猴爺（最高） | 50M 押注時 10K 寶石 | 1 | 一般獎勵格，僅造型為金邊框 + 猴爺圖 |
| 聚寶盆（次高） | 50M 押注時 8K 寶石 | 1 | 一般獎勵格，僅造型為金邊框 + 聚寶盆圖 |
| 紫底六角（高） | 50M 押注時 1K～6K 寶石 | 6 | 含 4 個大格（3K×2、4K、6K）+ 2 個小格（1K、2K） |
| 綠底圓形（中） | 50M 押注時 1K～2K 寶石 | 12 | |
| 紅底圓形（低） | 50M 押注時 1K 寶石 | 2 | |

## 視覺識別

| 元素 | 視覺特徵 |
|------|----------|
| 轉盤形狀 | 橢圓形（水平方向較寬），22 格沿橢圓邊緣排列 |
| 轉盤上方 | 猴爺圖 + 金邊寶石數量框（如「10K」） |
| 轉盤下方 | 聚寶盆圖 + 金邊寶石數量框（如「8K」） |
| 高獎勵格子 | 紫底六角形，尺寸較大 |
| 一般格子（綠） | 綠底/青底圓形 |
| 一般格子（紅） | 紅底/暗紅底圓形 |
| 資產欄位 | 左側金幣圖標 + 數值 / 右側寶石圖標 + 數值，常駐畫面上方 |
| 押注區 | 「押注」標籤 + 金幣圖案 + 數值 + 左右藍色 +/- 按鈕 |
| 背景 | 中式宮殿風格，深紅色調，兩側紅色燈籠裝飾 |
| 中獎格子效果 | 選中格子發出光暈/煙火特效 |
| 寶石贏取提示 | 右上角寶石欄位旁顯示「+X,000」浮動數字 |
