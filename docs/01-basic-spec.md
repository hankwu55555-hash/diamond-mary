# 01 · 基礎規格

## 系統概述

「寶石小瑪莉」是一款嵌入於交誼廳的小瑪莉輪盤遊戲。玩家使用金幣押注，透過輪盤轉動獲取寶石獎勵。遊戲沿用傳統小瑪莉的轉盤版面設計，上方為猴爺圖搭配寶石數量框、下方為聚寶盆圖搭配寶石數量框、四角為紫底格、其餘為紅底或青底格。每日可遊玩次數依玩家的 VIP 等級與所在國家而定。

## 核心規則表

| 規則項目 | 說明 |
|----------|------|
| 入口位置 | 玩家從【交誼廳】的【獲取寶石】頁籤點入 |
| 押注機制 | 使用 +/- 按鈕循環切換押注額；最大額按 + 回到最小額，反之亦然 |
| 轉盤連動 | 切換押注額時，轉盤上各格子的寶石數字同步變化，並依位數縮放字體大小 |
| 遊玩流程 | 點擊【開始遊玩】→ 金幣立即扣除 → 轉盤轉動 → 停止後顯示結果 → 點擊【領取】獲得寶石 |
| 自動旋轉 | 長按【開始遊玩】約 2 秒，跳出選擇面板（最多 4 個選項），選擇後自動遊玩 |
| 自動旋轉領取 | 自動旋轉期間不需按【領取】，寶石自動飛入 |
| 非無限自動旋轉 | 選擇非【無限】選項時，剩餘手數每手遞減 |
| 次數歸零 | 可遊玩次數歸零時，【開始】按鈕壓暗不可點，進入冷卻倒數；+/- 按鈕同時壓暗 |
| 遊玩中操作限制 | 轉盤轉動期間，轉盤區域、押注區、【開始遊玩】按鈕不可點擊 |
| 隨時關閉 | 遊玩過程中可點擊【關閉】直接關閉交誼廳畫面 |
