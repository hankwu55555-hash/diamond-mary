## 動畫規格

### 轉盤轉動動畫

| 項目 | 規格 |
|------|------|
| 觸發時機 | 玩家點擊【開始遊玩】後 |
| 動畫類型 | 轉盤格子依序高亮閃爍（沿用舊有小瑪莉規格） |
| 停止方式 | 系統決定結果後，轉盤減速停在選中格子 |
| 時長 | **TBD（沿用舊有規格）** |

### 格子壓黑效果

| 項目 | 規格 |
|------|------|
| 觸發時機 | 轉盤停止確定結果後 |
| 效果 | 未選中格子壓暗，選中格子維持高亮 |
| 用途 | 強調中獎格子的視覺焦點 |

### 寶石飛入動畫

| 項目 | 規格 |
|------|------|
| 觸發時機 | 手動模式：玩家點擊【領取】後；自動模式：自動觸發 |
| 動畫路徑 | 寶石從選中格子位置飛往畫面上方右側的寶石資產欄位 |
| 飛入後 | 寶石欄位數字向上 Count（數字跳動增加效果） |
| 時長 | **TBD（由前端/美術確認）** |

### 押注額切換動畫

| 項目 | 規格 |
|------|------|
| 觸發時機 | 玩家按 +/- 切換押注額 |
| 效果 | 轉盤各格子數字同步變化，字體大小依位數縮放 |
| 參考 | 有參考影片（見原始規格書） |

## 狀態視覺

### 冷卻倒數狀態

| 項目 | 視覺表現 |
|------|----------|
| 【開始】按鈕 | 文案變更 + 壓暗 + 顯示倒數時間 |
| +/- 按鈕 | 壓暗不可點 |
| 轉盤 | 維持靜態顯示 |

### 自動旋轉狀態

| 項目 | 視覺表現 |
|------|----------|
| 按鈕文案 | 依選擇變化，顯示剩餘手數或【無限】標記 |
| 寶石飛入 | 自動執行，無需等待玩家操作 |
| 停止按鈕 | 顯示 STOP 文案，玩家可點擊停止 |
