## 快速摘要

### 核心數值表

| 項目 | 數值 |
|------|------|
| 遊戲類型 | 小瑪莉輪盤（Slot-wheel） |
| 貨幣投入 | 金幣 |
| 貨幣產出 | 寶石 |
| 押注切換 | 支援 +/- 循環切換，金幣圖案 + 最多 9 位元 |
| 資產顯示上限 | 金幣/寶石各 15 位元（超過 12 位元縮小顯示） |
| 每日可遊玩次數 | 依 VIP 等級與國家而異（**TBD，由數值企劃設定**） |
| 自動旋轉選項 | 最多 4 個選項，含「無限」選項 |
| 支援語系 | EN、zh-TW、zh-CN、JA、ID、VI、TH（共 7 語） |

### 核心規則列表

1. 玩家從【交誼廳】→【獲取寶石】頁籤進入
2. 使用金幣押注，轉盤結果決定獲得的寶石數量
3. 押注額可透過 +/-
