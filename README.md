## 快速摘要

### 核心數值表

| 項目 | 數值 |
|------|------|
| 遊戲類型 | 小瑪莉輪盤（橢圓形，燈號跑動） |
| 貨幣投入 | 金幣 |
| 貨幣產出 | 寶石 |
| 押注額選項 | 50,000,000 / 100,000,000 / 500,000,000 |
| 押注切換 | 支援 +/- 循環切換，金幣圖案 + 最多 9 位元 |
| 資產顯示上限 | 金幣/寶石各 15 位元（超過 12 位元縮小顯示） |
| 每日可遊玩次數 | VIP1(銅)=0, VIP2=0, VIP3=20, VIP4=50, VIP5=100, VIP6(鑽石)=150 |
| 自動旋轉選項 | 10 / 50 / 100 / ∞ |
| 轉盤格子獎勵（50M 基準） | 1K～10K 寶石，隨押注額等比縮放 |
| 支援語系 | EN、zh-TW、zh-CN、JA、ID、VI、TH（共 7 語） |

### 核心規則列表

1. 玩家從【交誼廳】→【獲取寶石】頁籤進入
2. 使用金幣押注，轉盤結果決定獲得的寶石數量
3. 押注額可透過 +/- 按鈕循環切換
4. 轉盤停止後需點擊【領取】才能獲得寶石（自動旋轉模式除外）
5. 長按【開始】約 2 秒可啟動自動旋轉
6. 每日可遊玩次數受 VIP 等級與國家限制，歸零後進入冷卻倒數
7. 遊玩過程中可隨時點擊【關閉】離開
